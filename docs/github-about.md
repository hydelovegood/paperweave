# GitHub About

## Short Description

Local-first paper library CLI for PDF ingestion, structured summaries, deep Q&A, and forward-citation tracking.

## Longer About

PaperWeave（溯源文库） helps researchers turn a folder of PDFs into a structured local paper library with parsing, summaries, reviewer-style Q&A, citation tracking, and Markdown exports.
